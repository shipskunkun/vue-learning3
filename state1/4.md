在写组件中，首先开启， box-sizing: borderbox

```css
:root {} 根选择器

var() css属性函数， 必须以 --开头
:root {
  --main-bg-color: coral;
  --main-txt-color: blue;
  --main-padding: 15px;
}
 
#div1 {
  background-color: var(--main-bg-color);
  color: var(--main-txt-color);
  padding: var(--main-padding);
}

em: 所以它也是相对长度单位。


.g-button:focus{  //设置元素周围的轮廓，去掉默认的focus 时候的样式
	outline: none 
} 
```


npm i -D parcel-bundler  // -D 代表developer，面向开发者的包
