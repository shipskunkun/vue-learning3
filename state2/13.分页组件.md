组件内修改props 数据，

方法一：使用sync

```vue
// 父组件
<dialog-apply :visible.sync="dialogApplyVisible" />

// 子组件
<el-dialog
      :visible.sync="visible"
      title="申请"
      :before-close="onClose"
>

onClose() {
  this.$emit('update:visible', false)
}
```

方法二：

props 数据，赋值给data 或者computed， emit 方法，传递出去修改后的数据





data 和 computed 区别

```
当依赖的数据变化，就会重新执行
```
